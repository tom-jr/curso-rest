            Logo apos criar o projeto uma estrutura de diretorios é criada
 Na raiz pode se usar o arquivo mvn para rodar os comando marven de enpacotamento, build e etc.
  na raiz do projeto basta utilizar o 

  comando_ [./mvn {comand_maven}]

  comando_ [./mvn package]  impacota o projeto em um jar do diretorio_ [target]

  Apos a geração do Jar posso rodar o mesmo com o seguinte comando

  comando_ [java -jar {diretorio/arquivo.jar}]

  comando_ [./mvn clean]  limpa os builds Jars que existiren em target
